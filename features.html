Strings

Strings are positioned on top of frets.This makes it easier to press and have a sense of location on the fretboard. The notes are quantized exactly like a guitar and by default the strings are set to be the lower 3 strings,but this can be changed.

Picks and Knobs

Picks can detect the pressure that the user exerts and therefore are responsive to the playing. The 3 knobs can be mapped to any control parameter that the user wishes.

Proximity Sensor a.k.a Theremin

The proximity sensor can be enabled in legato mode to do pitchbends and achieve Theremin like effect.

Joystick

The buttons on the joystick module allow the user to switch between playing modes and enable/disable the proximity sensor. The X-axis of the joystick can be used to control modulation and the Y-axis can be used to control pitchbend. The pitchbend can be disabled by pressing the joystick.

Keypad Controller

Half of the keypad acts as a seperate instrument. That means, there are 16 slots open to assign samples sounds etc. The ideal use would be to configure it as a drum rack. The other half allows further control of a software instrument or DAW. The controls can be assigned by the user.

Picks and Knobs

Sliders

The sliders can be assigned to control any parameter as the user desires.
